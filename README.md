# 🤣 Random Joke Generator
A fun and simple **Random Joke Generator** that fetches jokes from an API and displays them dynamically on the screen. <br>Get ready to laugh with every click! 😂<br>
Deployed Link: https://random-joke-generator-by-jiya.netlify.app/

---

## 🚀 Features
- Fetches random jokes from **JokeAPI**.
- **Smooth fade effect** for better user experience.
- **Responsive design** for all devices.
- **Click button to generate a new joke**.

---

## 📌 How to Use
1. Open the **Random Joke Generator** in your browser.
2. A random joke is displayed automatically on load.
3. Click the **"Get Random Joke"** button to fetch a new joke.
4. Enjoy unlimited fun and laughter! 🎉

---

## 🛠️ Tech Stack
- **HTML** - Structure of the webpage
- **CSS** - Styling and animations
- **JavaScript** - Handles API requests and interactivity
- **JokeAPI** - Source of random jokes
